## Overview

## Installation
### Install [@NodeJS](https://nodejs.org/)

[Download](https://nodejs.org/) the recommended LTS version to ensure stability.

### Install npm
```bash
    cd Projet_SteamFlix-main
    npm install
```

## Modify
To watch and compile SASS files on changes

```bash
    npm run sass
```
